# Competition 2 - 系统设计大赛 (System Design Competition)

## 简介 (Introduction)
这是第二个计算机设计大赛，专注于系统架构设计和软件工程实践。

## 比赛规则 (Competition Rules)
- 时间限制：每题3小时
- 要求：完整的系统设计方案
- 题目数量：2题
- 评分标准：架构合理性 + 可扩展性 + 代码质量

## 题目列表 (Problem List)
1. 分布式缓存系统设计 (Distributed Cache System Design)
2. 微服务架构设计 (Microservices Architecture Design)

## 目录结构 (Directory Structure)
- `problems/` - 题目描述和需求
- `solutions/` - 参考设计方案
- `docs/` - 文档和说明
