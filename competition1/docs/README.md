# Competition 1 - 算法设计大赛 (Algorithm Design Competition)

## 简介 (Introduction)
这是第一个计算机设计大赛，专注于算法设计和数据结构优化。

## 比赛规则 (Competition Rules)
- 时间限制：每题2小时
- 语言：支持C++、Java、Python
- 题目数量：3题
- 评分标准：正确性 + 时间复杂度

## 题目列表 (Problem List)
1. 数组排序优化 (Array Sorting Optimization)
2. 最短路径算法 (Shortest Path Algorithm) 
3. 动态规划问题 (Dynamic Programming Problem)

## 目录结构 (Directory Structure)
- `problems/` - 题目描述和测试用例
- `solutions/` - 参考解答
- `docs/` - 文档和说明
