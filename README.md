# Computer-Design-Competition
这是两个计算机设计大赛的代码

## 项目概述 (Project Overview)

本仓库包含两个不同类型的计算机设计大赛，分别专注于算法设计和系统架构设计。

## 大赛介绍 (Competitions)

### 1. 算法设计大赛 (Algorithm Design Competition)
**目录**: `competition1/`

专注于算法设计和数据结构优化的编程竞赛，包含以下题目：
- 数组排序优化
- 最短路径算法
- 动态规划问题

**特点**:
- 时间复杂度和空间复杂度要求严格
- 支持 C++、Java、Python 等主流编程语言
- 重视代码效率和正确性

### 2. 系统设计大赛 (System Design Competition)
**目录**: `competition2/`

专注于系统架构设计和软件工程实践的设计竞赛，包含以下题目：
- 分布式缓存系统设计
- 微服务架构设计

**特点**:
- 强调系统架构的合理性和可扩展性
- 要求完整的设计方案和原型实现
- 重视工程实践和最佳实践

## 目录结构 (Directory Structure)

```
Computer-Design-Competition/
├── README.md                    # 项目总览
├── competition1/                # 算法设计大赛
│   ├── docs/                   # 大赛文档
│   ├── problems/               # 题目描述
│   └── solutions/              # 参考解答
└── competition2/               # 系统设计大赛
    ├── docs/                   # 大赛文档
    ├── problems/               # 题目描述
    └── solutions/              # 参考设计方案
```

## 如何参与 (How to Participate)

1. 选择感兴趣的大赛类型
2. 阅读对应目录下的 `docs/README.md` 了解比赛规则
3. 查看 `problems/` 目录下的题目描述
4. 参考 `solutions/` 目录下的解答示例
5. 提交你的解决方案
